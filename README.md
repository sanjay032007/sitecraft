# SiteCraft - Premium Web UI Templates 🎨

15+ production-ready web templates with Modern, Glassmorphism, and 3D interactive designs.

## Live Demo

👉 **[View All Templates](https://sitecraft-ui.vercel.app)**

## What This Is

SiteCraft is a collection of premium web templates designed for:
- Corporate websites
- Portfolio sites
- SaaS landing pages
- Creative agencies
- Startup showcases

Pick a template → Customize → Deploy. Done.

## Tech Stack

| **Styling** | Tailwind CSS v4 |
| **3D Graphics** | Three.js + React Three Fiber |
| **Animations** | Framer Motion |
| **Backend** | Supabase (optional) |
| **Language** | TypeScript |
| **Deployment** | Vercel + Netlify |
