# SiteCraft

SiteCraft is a modern, high-performance web application built with Next.js, React 19, Tailwind CSS v4, and Three.js.

## Tech Stack

- **Next.js (v16.2.7)**
- **React (v19)**
- **Three.js & React Three Fiber**
- **Tailwind CSS (v4)**
- **Framer Motion**
- **Supabase**
- **TypeScript**

## Getting Started

First, install the dependencies:
`ash
npm install
`

Then, run the development server:
`ash
npm run dev
`

Open http://localhost:3000 with your browser to see the result.
